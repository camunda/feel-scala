### Literal

Creates a new numeric value. Leading zeros are valid.

```js
1

0.5
.5

-2 

01

-0002
```

### Addition

```js
2 + 3
// 5
```

### Subtraction

```js
5 - 3
// 2
```

### Multiplication

```js
5 * 3        
// 15
```

### Division

```js
6 / 2  
// 3
```

### Exponentiation

```js
2 ** 3   
// 8
```
